# EnhancerPro
Image Enhancer is a web-based application that improves the quality of images using advanced image processing techniques. It allows users to upload photos and enhance brightness, contrast, sharpness, and clarity. Built with Python, Flask, and Pillow, the tool helps transform low-quality images into clearer and visually appealing results.
# Image Enhancer

Image Enhancer is a web-based application that improves the quality of images using modern image processing techniques. The application allows users to upload images and enhance them by improving clarity, brightness, contrast, and sharpness to produce better visual results.

---

## Features

* Upload images from your device
* Enhance image quality automatically
* Improve brightness and contrast
* Sharpen image details
* Simple and user-friendly interface
* Fast image processing

## Installation

### 1. Clone the repository

git clone https://github.com/your-username/image-enhancer.git

### 2. Go to project directory

cd image-enhancer

### 3. Install frontend dependencies

npm install

### 4. Install Python dependencies

pip install -r requirements.txt

### 5. Run the application

Start backend:

python main.py

Start frontend:

npm run dev

---

## Usage

1. Open the application in your browser
2. Upload an image
